##  Overview

**GRAPHYMA** is a modern, high-impact web platform designed to present Earth Observation consulting services in a clear, accessible, and visually engaging way.

The website reflects a vision where advanced satellite-based technologies are translated into practical, actionable solutions for businesses, institutions, and research communities.

---

## Key Features

* Fast and optimized build using Vite
*Clean, professional UI with responsive design
* Motion-driven experience powered by Framer Motion
*  Component-based architecture using React
*  Fully deployable as a static site (no backend required)

---

## 🛠️ Tech Stack

* **Frontend:** React
* **Build Tool:** Vite
* **Animation:** Framer Motion
